## Repositorio para código de búsqueda de contraejemplos de Selection Monotonicity de distintos algoritmos de apportionment

- "Brewer.jl" tiene un Pluto notebook para ejecutar desde la terminal con el comando Pluto.run() y buscar contraejemplos con el algoritmo de Brewer.
- "Contraejemplo Poisson Conditional Rounding.jl" tiene un Pluto notebook para ejecutar desde la terminal con el comando Pluto.run() y buscar contraejemplos con el algoritmo de Conditional Poisson Rounding.
- Contraejemplo.ipynb tiene un Colab notebook para ejecutar desde Google Colab y buscar contraejemplos con el algoritmo de Conditional Poisson Rounding.
